## 👋 Who we are

**Nemi Digital** is a small, senior team based in **Bundelkhand (Jhansi), Uttar Pradesh**, building fast, modern websites and growth systems — no agency bloat, no junior handoffs. You talk directly to the people writing the code and pushing pixels.

> *"We got tired of watching local businesses get sold bloated, slow, overpriced websites by big-city agencies. So we started Nemi."*

## 🚀 What we build

- ⚡ **Fast marketing & brochure sites** (Astro, Next.js, plain HTML — whatever fits)
- 🛒 **E-commerce stores** (Supabase / Shopify — see [Bundelkhand Jewellery](https://bundelkhandjwelery.netlify.app))
- 🧰 **Internal tools & admin dashboards** (product management, lead CRM, automated mailers)
- 🎨 **Branding & identity** (logos, color systems, brand guides)
- 📈 **SEO & conversion optimization** (technical SEO, local SEO for Bundelkhand/Jhansi/Kanpur)
- ✉️ **Automation & transactional email** (Resend, GitHub Actions cron, CSV mailers)

## 📂 This repo

This repo powers our **public website**:

| | |
|---|---|
| 🧑‍💻 **Stack** | Astro 4 + Tailwind CSS + Netlify |
| 🌐 **Live URL** | *(Deploying soon — see Netlify badge above)* |
| 📨 **Contact form** | Netlify Forms (zero-config) → optional Resend email when we add a custom domain |
| 📄 **Legal** | `/privacy` & `/terms` built in |
| 🗺️ **Sitemap** | Auto-generated at `/sitemap.xml` |

Run locally:

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # → ./dist (deploy target)
```

See [`README-site.md`](./README-site.md) for full deploy/configuration docs.
